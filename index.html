Colorwheel

A color picker coded using Raphael and jQuery. The wheel is drawn using the Raphael js library, and jQuery is used to assist with events.

The wheel can be resized, since it is drawn on the fly. It supports a small set of callbacks.

Usage and methods

var cw = Raphael.colorwheel(dom_element, height_width);
The wheel is drawn into a square, so only one dimension is required
cw.input(dom_element)
Sets a text input for setting and receiving the color as a hex value
cw.onchange(callback)
Set a callback for any change received. Callback takes one argument, a Rapahel RGB color object.
cw.ondrag(start, stop)
Set a start and stop callback for dragging. Callbacks takes one argument, a Rapahel RGB color object.
cw.color()
Get the current color as RGB object
cw.color(hex_value)
Set the current color via a hex value

Input

Assigning an input to a colorwheel will allow the user to directly change the hex color value.

When the input is set the colorwheel will set its color to the inputs value

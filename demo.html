Colorwheel Demo

var cw = Raphael.colorwheel(jQuery, height_width);
cw.input(DOMElement)
Sets a text input for setting and receiving the color as a hex value
cw.onchange(callback)
Set a callback for any change received. Callback takes one argument, a Rapahel RGB color object.
cw.ondrag(start, stop)
Set a start and stop callback for dragging. Callbacks takes one argument, a Rapahel RGB color object.
cw.color()
Get the current color as RGB object
cw.color(hex_value)
Set the current color via a hex value
